Install the Linux system on Windows

發表於 2021-10

中文版

Step 1 啟用Windows Linux System

需要先啟用 Windows Linux Subsystem 功能。需在『開啟或關閉 Windows 功能中設定』，剛裝完畢後，需要重新開機。

$U = mgh $

Step 2 安裝Ubuntu

接著到Windows Store 裡就有 Ubuntu App 可以選擇安裝

安裝成功後，就可以在 Windows 選單找到 Ubuntu App

  • 啟動Ubuntu後卻發現出現錯誤訊息…..

    這表示Windows-Subsystem-Linux (WSL)還沒打開，因此可以用接下來的方法解決。

Step 3 開啟PowerShell

輸入以下指令:

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

Step 4 啟動Ubuntu

安裝成功！

原本的 C/D/F 槽都 mount 在 /mnt 下

輸入cd /mnt/c/Users/*Name/Desktop

*Name: 根據自己的電腦名稱去修改

Step 5 更新套件

輸入 sudo apt-get update

Done!

English version

Step 1. Turn on Windows Linux Subsystem

  1. Press “Windows” + “R” keys simultaneously to open the Run prompt.
  2. Type in “Control Panel”
  3. Click on the “Programs and Features” option.
  4. Select the “Turn Windows Features On or OFF” option in the left pane.
  5. Scroll down and select “Windows Subsystem for Linux“
  6. 6.Click on “OK” to enable the feature.

Step 2 Install Ubuntu

  1. Open the Microsoft Store from the Start menu, and search for “Ubuntu” in the store
  2. To install , click it, and then click the “Get” or “Install” button to install it like any other Store application.

Step 3 Turning WSL ON Through PowerShell

  1. Search for PowerShell, right-click the top result, and select the Run as administrator option.

  2. Type the following command to enable the Linux subsystem and press Enter:

    Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

    Type Y and press Enter to confirm and restart your computer.

Step 4 Open Ubuntu

  1. Open Start, and click it .

It works!

Step 5 Update

This is the first time you are using your system after installing, you should run the update command: sudo apt-get update

Done!
